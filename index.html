ℹ️ 关于 CMX 阅读器

📖 阅读器介绍

CMX Reader v1.0 是一款专为 CMX 漫画格式打造的 Web 端阅读器，无需安装任何软件，直接在浏览器中即可流畅阅读 CMX 漫画文件。

  • 支持单页 / 双页模式，适配不同阅读场景
  • 支持 LTR（从左到右）和 RTL（从右到左）阅读方向
  • 内置缩略图导航，快速定位页面
  • 支持缩放、全屏沉浸式阅读
  • 支持 AES-256-GCM 加密文件的解密阅读
  • 内置 CMX 转换器，可将 ZIP/CBZ/图片文件夹转换为 CMX 格式
  • 完全在浏览器中运行，数据不上传服务器，保护隐私

📦 CMX 格式介绍

CMX（Comic eXchange） 是一种专为漫画设计的二进制文件格式，具有以下特性：

  • O(1) 随机访问 — 通过索引表直接定位任意页面，无需顺序解压
  • 多种压缩 — 支持 zlib / zstd / lz4 压缩算法
  • 内置缩略图 — 缩略图嵌入文件，阅读时可快速预览
  • AES-256-GCM 加密 — 军事级加密保护漫画内容
  • 元数据支持 — 内嵌标题、作者、日期等信息
  • 完整性校验 — 内置 MD5 校验，防止文件损坏
  • 多图片格式 — 支持 JPEG、PNG、WebP、AVIF

格式版本: CMX v1.0 | 魔数: CMX! / CMX~

📃 技术信息

  • 版本: v1.0
  • 技术栈: 纯 HTML/CSS/JavaScript，零依赖后端
  • 加密实现: Web Crypto API (AES-256-GCM)
  • 完整性校验: 纯 JS MD5 实现
  • 解压引擎: pako (zlib)
  • 转换器: JSZip
  • 兼容浏览器: Chrome 80+, Edge 80+, Firefox 80+
